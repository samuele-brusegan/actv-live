## Requisiti
- Node.js >= 20.x
- Composer >= 2.9.5

## Configurazione Locale
Per configurare il progetto, è necessario creare un file .env con le seguenti variabili:

```
DB_HOST=
DB_USER=
DB_PASS=
DB_NAME=
```

Eseguire i seguenti comandi per installare le dipendenze:
```bash
npm install
composer install
```
